📖 사용법 안내

1. HTML에 라이브러리 추가

<!-- CMS 연동 스타일 --> <link rel="stylesheet" href="css/misopin-cms.css"> <!-- CMS 연동 스크립트 --> <script src="js/misopin-cms.js"></script>

2. 게시판 목록 표시

<!-- HTML --> <div id="notice-list"></div> <script> // 공지사항 불러오기 misopinCMS.renderBoardList('notice-list', { type: 'NOTICE', limit: 5 }); // 이벤트 불러오기 misopinCMS.renderBoardList('event-list', { type: 'EVENT', limit: 10 }); </script>

3. 수동 API 호출

// 게시글 가져오기 const posts = await misopinCMS.getBoardPosts({type: 'NOTICE'}); // 특정 페이지 가져오기 const page = await misopinCMS.getPage('about'); // 활성 팝업 가져오기 const popups = await misopinCMS.getActivePopups();

4. 자동 팝업 표시

페이지 로드 시 CMS에서 설정한 팝업이 자동으로 표시됩니다. 사용자가 "오늘 하루 보지 않기"를 선택하면 24시간 동안 표시되지 않습니다.
